# Task-Manager
A Task Manager is a software application used to create, organize, track, update, and manage daily tasks. It helps users improve productivity by keeping track of pending and completed activities.

HTML

HTML (HyperText Markup Language) is used to create the structure and content of the Task Manager web application, such as forms, buttons, task lists, headings, and input fields.

CSS

CSS (Cascading Style Sheets) is used to design and style the application. It controls colors, layouts, fonts, spacing, animations, and responsive design.

JavaScript

JavaScript is used to add interactivity and dynamic functionality to the Task Manager. It handles operations such as adding tasks, editing tasks, deleting tasks, searching, filtering, and updating task status.

Python

Python is used as the backend programming language. It processes user requests, handles application logic, and communicates with the database.

Flask

Flask is a lightweight Python web framework used to connect the frontend of the Task Manager with the backend and database.

SQLite

SQLite is a lightweight relational database used to permanently store task information such as task title, description, priority, category, due date, and completion status.

Task Priority

Task Priority indicates the importance of a task. Tasks can be classified as High, Medium, or Low priority, helping users focus on important work first.

Task Filtering

Task Filtering allows users to display specific tasks based on conditions such as All Tasks, Active Tasks, Completed Tasks, or Priority.

Conclusion

The Task Manager System provides a simple and efficient way to organize daily activities. By using HTML, CSS, JavaScript, Python, Flask, and SQLite, the application provides an interactive interface and reliable task management functionality.
